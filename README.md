## Stopwatch using ReactJS and Material-UI

This project is to get hands on ReactJS and Material-UI. So I created a simple **Stopwatch** with functionality to create laps/splits.

### Actions Available

- Play or Pause the counter
- Create Laps
- Reset the Stopwatch
